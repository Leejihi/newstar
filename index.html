HTML

HTML: Hypertext Markup Language

  • 웹을 이루는 가장 기초적인 구성
  • 웹페이지와 그 내용을 구조화하기 위해 사용하는 코드로 마크업 언어로 이루어져 있다.
    마크업 언어는 태그로 이루어져 있다. HTML은 태그로 이루어졌다.
  • 태그는 요소(elament), 속성(attribute), 값(value)으로 이루어져 있다
  • HTML의 태그는 크게 블록 요소와 인라인 요소로 나누어 진다.

HTML 태그 = 블록요소 + 인라인요소

구분 블록요소 block element 인라인요소 inline element
정의 해당 태그가 브라우저에서 독립적인 하나의 공간을 가지는 요소들. 해당 태그가 마크업하고 있는 컨텐츠의 크기만큼만 공간을 차지하여 좌우로 다른 태그가 나란히 위치하는 요소들.
특징
  • 기본적으로 가로폭 전체의 넓이를 가지는 직사각형 형태

  • 블록요소는 모든 인라인요소를 포함할 수 있고 다른 블록요소도 일부 포함 할 수 있다.

  • 블록요소 다음에는 줄바꿈이 생긴다.
  • 기본적으로 컨텐츠가 끝나는 지점까지를 넓이로 가진다.
    그래서 임의로 width, height로 변형을 줄 수가 없고, margin, psdding 속성은 좌우 간격만 반영되고 상하간격은 반영되지 않는다.

  • 레이아웃에 영향을 주지않고, 글자에 꾸밈을 줄 수 있다.

  • 중첩요소: 인라인요소는 항상 블록요소안에 포함되어 있으며 인라인요소안에 다른 인라인요소가 포함될 수 있다.

  • 인라인요소 다음에는 줄바꿈없이 한줄에 나란히 배치된다.
CSS width, height, margin, padding 등을 사용하여 형태를 변형하여 레이아웃을 수정할 수 있다. line-height로 줄의 높낮이를 조절할 수 있고 text-align으로 텍스트의 중앙, 좌,우측 정렬을 할 수 있다.
CSS 명령어: display:inline
블록요소를 인라인 요소의 속성으로 변경할 수 있다.		 CSS 명령어: display:inline / display:inline-block
블록요소를 인라인 요소의 속성으로 변경할 수 있다.
